Automated Image Restoration using Generative Adversarial Networks with Synthetic Data Training and Image Quality Preservation

Image restoration is an issue which aims to recover parts of an image that have degraded significantly making manual re-touching tools infeasible to use due to the volume and variation of the damage. Proposed is a new automated process of image restoration based on Generative Adversarial Networks which is transferable, efficient and minimises information loss.

May 2022
  • Image Segmentation Framework.
  • Creation of 7 Synthetic Deep Learning Datasets.
  • Applying Datasets to train Generative Adversarial Network.
  • Restoring Images present with Real (Non-Synthetic) Damage.

A Dynamic Navigation System Simulation

Satnavs, as accurate as they are, are not always able to tell the user in real time about oncoming accidents, therefore say if there is a car crash ahead, a normal satnav would not be able to tell the driver. However, a self-driving car would be able make their own decision and re-direct if necessary, to get to their destination as quick as possible. This is usually done by tapping into news feeds, that indicate to them about traffic and incidents on the road and detecting minor changes in the traffic around the car.

June 2019
  • Dynamic Re-routing at node points (junctions).
  • Traffic Simulation with cars taking random routes mimicking congestion at given different roads.
  • Can be applied to various road network configurations.
  • Outlines the benefit of using a social network between cars to improve routing between two locations.
