Lottery Web App
November 2020
Project Description
Over the past four weeks, I have developed a web application that allows a user to create an account, log in and enter custom or “lucky dip” lottery numbers and check if they are a winner. This is done through Java Servlets, JSP pages and a database backend all deployed on a Tomcat server.
Handling data about people or important entities must perform securely to protect its contents, otherwise, it opens the opportunity for malicious attacks. My application used many different security techniques such as encryption, hashing and SQL Injection/Path traversal prevention.
Before this project, I had close to no experience in HTML and JavaScript (JS). Through the project requirements, I learned how to write many different scripts such as input validation in JS and understood how similar the language is to Java which I have a lot of experience in already. I found it difficult understanding how sessions work between java servlets and 'jsp' pages however by the fourth week and some implementation of session management, I gained an understanding of how they operated.
I enjoyed learning how to encrypt and decrypt files as it is something I can transfer into my programming practices to create a reliable system with great security protocols.
